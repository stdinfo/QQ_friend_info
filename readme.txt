功能：
	DONE：批量爬取好友信息，包括基本信息和成为好友天数，并保存到数据库
	
用法：
	1：获取好友列表（由于比较懒，所以列表需要手动导出）。
		登陆网页版QQ邮箱。
		在左侧点击通讯录。
		上方工具-导出联系人导出（导出格式为vcard）（导出你要爬的所有人，一般就是全部好友）
			【tip:可以新建一个分组把要导出的好友全部扔进去，导出的时候导出这个分组即可。新建分组在右上方】
			【还是不会的话自行百度“批量导出qq好友”】
		将文件重命名为qq.vcf,放置在主程序根目录下。
	2：安装相关依赖环境
		python：3.X
		
		下面几个包直接pip install就可以
		selenium
		re
		pymysql
		
		chrome[如果不用chrome浏览器，那么需要下载单独的selenium驱动程序（chrome的已经在文件夹里面放好了），自行百度]
		
	3：配置数据库
		使用mysql数据库，远程或本地均可
		参数在运行时输入即可
		首次使用前，运行setupdb.py创建数据表
	
	4：运行主程序main.py
	
PS：
	1：多次执行，参数可以直接写到py文件里面，所有变量都在最前面。
